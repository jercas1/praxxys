Installation

1. composer install
2. rename .env.example to .env
3. php artisan key:generate
4. php artisan migrate
5. php artisan db:seed
6. php artisan storage:link
7. npm install
8. npm run dev
9. php artisan serve
10. go to http://127.0.0.1:8000/
